# IoT-Based Heart Attack Detection System

## Description
This project is an IoT-based health monitoring system designed to detect abnormal heart rate conditions and provide alerts in real-time. It helps in early detection of potential heart attacks by continuously monitoring heart rate data.

## Features
- Real-time heart rate monitoring
- Sensor-based data collection
- Alert system for abnormal conditions
- Continuous health tracking

## Components Used
- Heart Rate Sensor
- Microcontroller (Arduino)
- IoT Module

## Working
The system collects heart rate data using sensors and sends it to the microcontroller. If abnormal readings are detected, the system triggers an alert to notify users.

## Applications
- Healthcare monitoring
- Early heart attack detection
- Remote patient monitoring
